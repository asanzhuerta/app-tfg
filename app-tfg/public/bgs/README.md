# Fondos públicos

Imágenes de fondo usadas por landing, login y paneles por rol.

## Función

Dar identidad visual diferenciada a las experiencias de acceso, cliente, comercial y administración.

Los fondos se sirven públicamente desde Next.js y deben tener nombres estables para que los componentes puedan referenciarlos sin depender de rutas externas.

## Reglas de uso

- Mantener imágenes optimizadas para web.
- Evitar degradados embebidos si el componente ya controla capas visuales.
- No usar esta carpeta para imágenes de productos ni recursos de prueba; esos flujos van por Cloudinary.
