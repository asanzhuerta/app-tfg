# área cliente

Rutas protegidas para salones y clientes profesionales.

## Función

Dar acceso a panel cliente, perfil, catálogo, coloración, pedidos, promociones, formaciones, notificaciones y fichas técnicas del salón.

El cliente usa la app para operar su salón y consultar recursos, no para administrar el sistema. Esta carpeta mantiene esa experiencia separada.
