# Comunicaciones admin

Gestion administrativa de promociones, formaciones y rangos.

## Funcion

Crear comunicaciones comerciales y segmentarlas por cliente, rango, producto o linea comercial.

Las promociones usan tipos estructurados (`percentage_discount`, `volume_percentage_discount`, `gift_product`) y pueden guardar imagen y PDF adjunto. Los descuentos economicos se resuelven desde el servicio de pedidos; los regalos quedan como condicion comercial visible.
