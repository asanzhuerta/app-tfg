# Comunicaciones admin

Gestión administrativa de promociones, formaciones y rangos.

## Funcion

Crear comunicaciones comerciales y segmentarlas por cliente, rango, producto o línea comercial.

Las promociones usan tipos estructurados (`percentage_discount`, `volume_percentage_discount`, `gift_product`) y pueden guardar imagen y PDF adjunto. Los descuentos económicos se resuelven desde el servicio de pedidos; los regalos quedan como condición comercial visible.
