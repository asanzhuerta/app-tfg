# Area comercial

Rutas protegidas para usuarios con rol `commercial`.

## Funcion

Gestionar clientes asignados, visitas, rutas, pedidos, catalogo, coloracion, promociones, formaciones y actividad comercial.

## Por que existe

El comercial necesita una vista operativa enfocada en territorio, visitas y seguimiento diario. Esta carpeta agrupa esa experiencia.
