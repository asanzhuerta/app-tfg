# Servicios clientes

Operaciones de datos de clientes.

## Función

Consultar clientes, asignaciones, rangos, datos comerciales y estimaciones.

`client-tier-settings.ts` concentra la configuración de umbrales de compra, fecha/frecuencia de recálculo y ejecución manual de reasignación de rangos. Los rangos se mantienen jerárquicos: Plata, Oro y Platino.
