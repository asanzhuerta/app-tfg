# Documentacion LaTeX

Memoria tecnica y academica del TFG.

## Que contiene

- `main.tex`: documento principal.
- `sections/`: capitulos y apartados redactados.
- `figures/`: diagramas, capturas y figuras insertadas.
- `config/`: configuracion LaTeX reutilizable.
- `bibliography.bib`: referencias bibliograficas.

## Funcion

Mantener la documentacion del TFG sincronizada con el estado real de la aplicacion.

## Por que existe

La memoria no es un simple anexo: explica requisitos, diseno, implementacion, pruebas, arquitectura y decisiones tecnicas. Por eso vive separada del codigo pero dentro del mismo repositorio.

## Nota

Los ficheros generados por LaTeX (`.aux`, `.log`, `.toc`, `.lof`, `.lot`, `.fls`, etc.) son artefactos de compilacion y no representan contenido fuente.
