# Figuras y capturas

Diagramas, capturas reales y recursos visuales usados en la memoria.

## Funcion

Guardar las figuras insertadas en LaTeX: diagramas de arquitectura, datos, secuencia, despliegue y capturas funcionales de la aplicacion.

## Por que existe

La memoria necesita evidencia visual legible y versionada. Mantener las figuras aqui permite referenciarlas desde LaTeX sin mezclarlas con assets publicos de la aplicacion.
