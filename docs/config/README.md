# Configuracion LaTeX

Archivos de configuracion reutilizables para la memoria.

## Funcion

Centralizar ajustes de estilo, paquetes, comandos o parametros compartidos por el documento principal.

## Por que existe

Separar la configuracion evita que `main.tex` crezca demasiado y facilita mantener una estructura limpia cuando se ajusta formato, figuras o convenciones del documento.
