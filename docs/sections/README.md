# Secciones de la memoria

Capitulos y apartados parciales incluidos desde `main.tex`.

## Funcion

Dividir la memoria en unidades manejables: introduccion, requisitos, analisis, diseno, implementacion, pruebas, conclusiones y anexos.

## Por que existe

La modularizacion facilita revisar y ampliar la documentacion sin editar un unico archivo gigante.
